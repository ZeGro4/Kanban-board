# Kanban-board

2. Система управления задачами с ролями (Kanban-доска)
Что делает:
Проекты → Доски → Задачи.

Технологии:

ASP.NET Core MVC + Razor или Web API + отдельный Blazor/Frontend (Angular/React — но это уже fullstack)

Identity Framework (регистрация, логин, роли)

EF Core + миграции

Background service для напоминаний по задачам (IHostedService)
